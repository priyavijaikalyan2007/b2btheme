MetricCard

Single-value KPI card for dashboard strips. Four lifecycle states (ready / loading / error / empty), trend with intent override, optional inline sparkline, optional secondary text. Click semantics: href → <a>; onClick only → <button>. See ADR-129.

KPI strip (medium)

Compact strip (small) — for analytics rows with >6 cards

Lifecycle states

Loading → ready transition reserves the card height (min-height + visibility:hidden on the value cell), so resolving an async value never produces a layout jump.

Intent override

Trend direction drives the arrow; intent drives the colour. Decoupling them lets “errors increased” show an up-arrow but a negative-red palette, and “costs avoided” show a down-arrow with a positive-green palette.

Click semantics

First card has href set → root is an <a> (Enter activates, right-click shows browser link menu).
Second card has only onClick → root is a <button> (Enter / Space activate; not draggable).
