ChartPanel

Theme-aware Chart.js wrapper. Supports bar, line, area, and sparkline kinds. Reads window.Chart from a CDN script tag — the consuming application loads Chart.js separately, mirroring the CodeMirror / marked / KaTeX pattern (ADR-028, ADR-130). Re-themes automatically on data-bs-theme toggle.

Live refresh (same series IDs → no animation)

Click Refresh to swap data with the same series ID. The wrapper calls chart.update("none") so the change happens in a single frame — no animation jank on poll-driven dashboards. Rebuild issues a new series ID, forcing a full Chart.js rebuild.

Multi-series with intent

Each series carries an intent that maps to a --theme-* token (resolved at paint time via getComputedStyle). Toggle the theme to see re-paint without remount.
