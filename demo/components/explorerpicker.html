ExplorerPicker

All Components

Browse Mode (Single-Select)

Browse the org tree, expand folders, and select a resource. Ontology-driven icons, breadcrumb path on selection.

Multi-Select Mode

Checkboxes for selecting multiple resources. Pre-selected items highlighted on render.

Container Selection Mode

Only folders and org units are selectable. Assets are dimmed. Used for “Create from…” destination picking.

Search Mode

Type in the search bar to filter results. Breadcrumb paths under each result for disambiguation. Mock search API with 300ms debounce.
