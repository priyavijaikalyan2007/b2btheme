GraphCanvas

Interactive SVG graph visualization with zoom/pan, layout algorithms, selection, and keyboard shortcuts.

Multi-Edge Routing

When multiple edges connect the same node pair, curves fan out with perpendicular Bézier offsets and staggered labels.

Edge Label Density (hubs)

Hubs — nodes with many incident edges — used to produce an illegible smudge of overlapping labels. edgeLabelDensity collapses hub-incident labels to a 2–3 char abbreviation; hover for the full label. Non-hub edges keep their full text. Collision stepping also spaces labels along the edge.

Icon Name Resolution

Node icons accept BI class names, Feather names, or raw Unicode. The component resolves names to glyphs automatically.
